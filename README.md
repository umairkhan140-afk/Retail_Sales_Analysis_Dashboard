## Financial Sales Dashboard (Microsoft Excel)

## Overview 
Developed an interactive Excel dashboard by transforming raw retail sales data into meaningful KPIs, business insights, and actionable recommendation

Interactive KPI Dashboard 
Pivot Tables & Pivot Charts
Slicers for dynamic filtering 
Data Cleaning & Transformation
Business Insights & Recommendations
## Tools Used 
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
